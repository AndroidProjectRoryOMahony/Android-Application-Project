Welcome the The University of Manchester 3rd year project website. This simple website has been created to display information regarding the project as well as display and modify the contents of a database that contains information on modules measuring daa around the campus.

A description of the project can be found under the section named "The Project". As new modules are installed the "Add Equipment" section can be used to add all the details for the equipment into the database. Current database entries can be edited, for example the types of data types being measured, under the "Modify Database" section.

This project forms a small section of the overall UoM smart grid project. The database containing information on the data recording modules can be viewed using the button below.
