The project focuses on developing an Android application that will be used to connect with several data storage systems around The University of Manchester.


The application will download data from these systems and display the information graphically/textually, allowing trends and errors to be discovered.

The NI CompactRIO will be used to measure various data types around the campus. This data will be stored using PI servers from OSIsoft which will allow the application to access both live and historic data.


A database (accessible below) contains all the information on the currently installed cRIO systems around the university.
